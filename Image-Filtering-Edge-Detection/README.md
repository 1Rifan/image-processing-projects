## 🧪 Features

### 1. Image Filtering (Spatial Domain)
- Mean Filter
- Gaussian Filter
- Custom filtering function using convolution

### 2. Image Filtering (Frequency Domain)
- Fourier Transform (FFT)
- Low-pass filtering using a circular mask
- Image reconstruction using inverse FFT

### 3. Edge Detection Techniques
- Sobel Operator
- Laplacian of Gaussian (LoG)
- Canny Edge Detector
